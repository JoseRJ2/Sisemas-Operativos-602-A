Los números mágicos son secuencias de bytes ubicadas al principio de un archivo que identifican el tipo o formato del propio archivo. En los sistemas operativos Unix, los números mágicos son utilizados para determinar el tipo de archivo de forma precisa independientemente de su extensión. Por medio del comando "file" se puede examinar el contenido del archivo buscando las secuencias de bytes (números mágicos) luego consulta una base de datos interna para identificar el tipo de archivo en función de las firmas encontradas.

La base de datos utilizada por el comando file contiene entradas para una amplia variedad de tipos de archivo y sus correspondientes firmas de archivo. Estas entradas se actualizan regularmente para reflejar nuevos tipos de archivos y cambios en las firmas de archivo existentes. En resumen, los "números mágicos" en Linux son secuencias de bytes ubicadas al principio de un archivo que se utilizan para identificar su tipo o formato, y son ampliamente utilizados por herramientas como el comando file para determinar el tipo de archivo de forma precisa y confiable.
